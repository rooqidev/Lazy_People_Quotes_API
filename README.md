# Lazy People Quotes API

A simple backend API built with **FastAPI**, designed to serve random quotes for lazy people.  
This project is structured for learning and future deployment.

---

## Features
- RESTful API endpoints
- Lightweight FastAPI backend
- Easy to run locally
- Modular structure for learning and expansion

---

## Requirements
- Python 3.x
- FastAPI
- Uvicorn

Install dependencies:

```bash
pip install -r requirements.txt
